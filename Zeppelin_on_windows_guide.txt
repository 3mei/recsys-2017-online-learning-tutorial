Zeppelin on windows guide
	download java 8 - http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html

	download anaconda with python 3.6 - https://www.continuum.io/downloads#windows
		overwrite PATH environment variable - anaconda as default python interpreter

	download 0.7.0 Zeppelin - http://zeppelin.apache.org/download.html

	paste notebooks to notebook folder

	SPARK
		download winutils.exe - https://github.com/steveloughran/winutils/blob/master/hadoop-2.7.1/bin/winutils.exe
		create folder hadoop/bin somewhere
		create HADOOP_HOME system environment variable set it to ../hadoop
		Zeppelin --> interpreters --> spark --> zeppelin.spark.useHiveContext : set to false
		enjoy

	FLINK
		download jar from given source
		Zeppelin --> interpreters --> flink: 
			new Property: taskmanager.numberOfTaskSlots: 4
			new Dependency: path to downloaded jar

	work :D

	Difference between Win and Linux:
		you start the linux from the root and thw windows from the bin, so data_session should be in the directory, where you started
